## ğŸš€ Getting Started

To make it easy to run **Elevate-ED** on any 
operating system, I have provided setup scripts
located in the `/Run` directory. These scripts will
help you prepare the environment and launch
the application with a single command.

### How to Run
1. **Clone the repository:**
	```terminal
	cd ~
	git clone https://github.com/kararkm61/Elevate-ED.git
	cd Elevate-ED
	```
2. **Execute the script for your OS:**
	Navigate to the Run folder and 
	run the appropriate file:
	* **Linux**: use Linux.sh
	```bash
	bash Data/Run/Linux.sh
	```
	* **MacOS** (Darwin): use Darwin.zsh
	```zsh
	zsh Data/Run/Darwin.zsh
	```
	* **Windows** (win32): use Win32.cmd
	```cmd
	Data/Run/Win32.cmd
	```
	Note: These scripts will automatically 
	check for Python and handle 
	the initial setup for you.
